# Introduction
This is the python implementation of the simple rest api.

# Getting Started
Install dependencies:
```bash
make install
```

Run the application.
```bash
make run
```

Run the test suite:
```bash
make test
```

Create migrations from models:
```bash
make makemigrations
```

Migrate the database:
```bash
make migrate
```

Visit the swagger ui:
```bash
http://localhost:8000/swagger/
```
